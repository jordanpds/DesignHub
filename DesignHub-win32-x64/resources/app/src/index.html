OEM Apps

You can use DesignHub's quick launch to open most commonly used programs with the required settings for specific OEM's as well as also launching the native apps.

Ford Apps

Provides the most recent Catia v5 R29 applications setup with Ford Standards as well as means to login to Teamcenter without launching Catia.

Theorem

GM Apps

GM has their own launcher that they use to connect to Teamcenter as well as provide their own toolkit for NX. When launching anything that isn't the NX toolkit you will be asked to login. Once logged in you will be able to access Teamcenter as well as being able to launch other aspects of the GM Launcher.

Toyota Apps

Toyota uses Catia R27 as their native application. Launching the application through the DesignHub launcher will launch it with Toyota Standards already set. You can also enable Theorem which will allow you to save your file as a JT.

Theorem

Daimler Apps

Daimler uses a specific app called Daimler NX Startconfigurator that they use in place of Teamcenter. It allows you to select your business unit, department/group, and license bundle. Once all of those are selected you can hit Start NX and it will open NX 12 with the Daimler toolkit.

Native Apps

Native Apps provides all apps without the settings required for OEM versions of them.

Catia Apps

DesignHub allows you to launch each native CATIA version with or without Theorem (if applicable) without the OEM settings.

Theorem

NX Apps

DesignHub allows you to launch each native NX version with or without Theorem (if applicable) without the OEM settings.

Theorem

Other Apps

DesignHub allows you to launch other not commonly used applications.

Maintenance Apps

Here you can run some basic maintenance to clean up your system should you encounter any errors.

Clean Teamcenter

Cleaning Teamcenter will remove all temp files and require a resynchronization of Teamcenter when logged in.

CATIA Settings

Cleaning CATIA Settings will delete all pre-existing user settings for all CATIA versions and reset the standards to their default settings.
